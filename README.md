# ☕ BasicJavaConceitos

> Um Repositório Focado Em **Descomplicar A Sintaxe E A Lógica Base Do Java**, Com Explicações Simples, Passo A Passo E Comentários Direto No Código.

Este Projeto Nasceu Do Meu Processo De Aprendizado Em Java. Em Vez De Deixar Os Conceitos Espalhados, Reuni Os Fundamentos Da Linguagem Em Um Único Arquivo De Referência — Cada Linha Comentada, Cada Conceito Explicado No Contexto Em Que Aparece. A Ideia É Que Qualquer Pessoa (Inclusive Eu No Futuro) Consiga Bater O Olho E Relembrar Como Tudo Funciona.

---

## 🎯 Objetivo

Servir Como Um **Guia De Consulta Rápida** Para Quem Está Começando Em Java, Cobrindo Do Básico Do Básico Até A Porta De Entrada Da Orientação A Objetos — Sempre Com O "Porquê" Por Trás De Cada Conceito, Não Só O "Como".

---

## 📚 O Que Você Vai Encontrar

O Arquivo [`GuiaBasicoJava.java`](./GuiaBasicoJava.java) Cobre, Em Ordem Didática:

| # | Tópico | O Que É Abordado |
|---|--------|------------------|
| 1 | **Variáveis E Tipos Primitivos** | `byte`, `short`, `int`, `long`, `float`, `double`, `char`, `boolean` E `String` |
| 2 | **Operadores** | Aritméticos, De Comparação E Lógicos (`&&`, `||`, `!`) |
| 3 | **Condicionais** | `if` / `else if` / `else` E O `switch` Moderno (Java 14+) |
| 4 | **Arrays (Vetores)** | Coleções De Tamanho Fixo E Acesso Por Índice |
| 5 | **Collections (ArrayList)** | Listas De Tamanho Dinâmico |
| 6 | **Loops** | `for`, `for-each` E `while` Com Casos Práticos |
| 7 | **Casting** | Conversão Entre Tipos (*Widening* E *Narrowing*) |
| 8 | **POO** | Classes, Objetos, Construtores, Encapsulamento E Getters/Setters |

---

## 💡 Conceitos-Chave Destacados

Alguns Pontos Que Costumam Confundir Quem Está Começando — E Que Estão Explicados Com Cuidado No Código:

- **`==` Vs `.equals()`** → Por Que Comparar `String` Com `==` É Uma Armadilha Clássica.
- **`public static void main`** → O Que Cada Palavra Da Assinatura Significa E Por Que O `static` É Obrigatório.
- **Casting** → Por Que `(int) 3.14` Vira `3` (Descarta, Não Arredonda).
- **`for` Vs `for-each` Vs `while`** → Quando Usar Cada Um.

## 🚀 Como Executar

Você Só Precisa Do **JDK** Instalado (Versão 17 Ou Superior Recomendada).

```bash
# 1. Clone O Repositório
git clone https://github.com/SEU-USUARIO/BasicJavaConceitos.git

# 2. Entre Na Pasta
cd BasicJavaConceitos

# 3. Compile O Arquivo
javac GuiaBasicoJava.java

# 4. Execute
java GuiaBasicoJava
```

> 💡 O Nome Do Arquivo `.java` Deve Ser Igual Ao Da Classe Pública — Por Isso É `GuiaBasicoJava.java`.
